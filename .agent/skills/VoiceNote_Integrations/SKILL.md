# Bibliothèques et APIs

Les bibliothèques et API suivantes sont considérées comme MUST-HAVE pour le projet et doivent être utilisées comme standards de développement :

1. **React Native / Expo** : Framework de base, SDK Expo à jour, utilisation de `expo-router` requise.
2. **Supabase** : Base de données, Auth, Storage, Edge Functions, RLS.
3. **TailwindCSS / NativeWind** : Moteur de style pour React Native.
4. **TypeScript** : Mode strict obligatoire.
5. **Zustand** : State management.
6. **OpenAI API** : Modèles (Whisper pour audio fallback, GPT-4o pour IA textuelle).
7. **Deepgram API** : Transcription principale et diarization.
8. **RevenueCat** : Achats In-App et abonnements.
9. **React Native Reanimated** : Animations d'interface.
10. **react-i18next** : Internationalisation et gestion de langues.
11. **Google Calendar API** : Intégration agenda.
12. **Notion API** : Export de notes/transcriptions.
13. **Push Notifications (Expo)** : expo-notifications, push tokens.
14. **React Native SVG** : Rendu vectoriel (mind maps générées).
15. **Audio/Vidéo (expo-av)** : Enregistrement et lecture des sessions audio.
