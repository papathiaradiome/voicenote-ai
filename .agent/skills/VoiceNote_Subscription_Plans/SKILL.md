# VoiceNote Subscription Plans

## Plan GRATUIT (free)
- 5 enregistrements/mois
- 30 min transcription/mois
- Résumés basiques
- 1 dossier custom
- Pas de mind map, pas de chat IA, pas d'import YouTube, pas d'export PDF, pas de calendrier
- Import max 10Mo/10min
- Stockage 500Mo

## Plan PRO (pro)
- Tarif : 9,99€/mois ou 79,99€/an
- Enregistrements illimités
- 10h transcription/mois
- Résumés avancés
- Mind maps IA
- Chat IA 50 questions/mois
- Import YouTube
- Dossiers illimités
- Export PDF/DOCX
- Pas de calendrier, pas de recherche sémantique
- Import max 200Mo/2h
- Stockage 10Go

## Plan BUSINESS (business)
- Tarif : 24,99€/mois ou 199,99€/an
- Tout illimité
- Calendrier + enregistrement auto réunions
- Traduction
- Recherche sémantique IA
- Collaboration d'équipe
- Support prioritaire
- Templates de compte-rendu
- Import max 1Go/5h
- Stockage 100Go

## Règles d'intégration Paywall
- Quand une feature est bloquée : afficher un PaywallModal contextuel avec le nom de la feature, sa description, et un bouton « Passer à Pro » ou « Passer à Business »
- Badges visuels « PRO » ou « 🔒 » sur les features verrouillées dans l'UI
- Vérifier les limites AVANT chaque action premium (pas après)
